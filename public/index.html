Fácil Instalação

Potente e Intuitivo

Multiplataforma

Funciona nos principais sistemas operativos, sem necessidade de instalação de dependências!

Interface Amiga do Utilizador

Com uma interface simples e leve em recursos, o pyVoice não assusta o utilizador, nem requer grandes tempos de aprendizagem para ser utilizado.

Capacidade de Aprendizagem

Cada utilizador é diferente, e por isso o pyVoice aprende consigo! Se uma frase compatível com um comando for dita, pode escolher adicioná-la como trigger para esse comando. Simples!

Open Source

A privacidade é muito importante, e por isso, o pyVoice adotou a política Open Source, ou código aberto, para todos os seus componentes. Melhor ainda, todos podem ajudar a tornar o projeto melhor!
